HELPFUL RESOURCES
FOR WEB DESIGN

1.

GOOGLE FONTS

-Is a free-source website developed by Google to provide fonts.

-Part of improving the aesthetics of a design is selecting appropriate fonts.

-Fonts can convey all sorts of meanings and messages.

-With Google Fonts, you can gain free access to a wide source of fonts that can
suit your every need.

-It also conveniently provides you with the syntax needed to activate your fonts.

2.

IONICONS

-A website which provides icons for free use in multiple projects including HTML and CSS.

-Icons are an aesthetically pleasing, stylized way to add elements yo your design

-Icons can be used for easy understanding for the user in terms of application.
These can include: navigation, hyperlink, and interface controls.

-IONICONS provides you with a free resource of almost every conceivable icon
out there.

-IONICONS also provides the exact syntax you need to activate its codes.

3.

PINTEREST

-A photo sharing website.

-Most people use Pinterest as a source of aesthetic pictures.
This can be applied to Web Design

-Pinterest is a great source of ideas for designing websites.

-Although Pinterest cannot provide the source code for how to code certain interfaces, it gives you a means of reverse-engineering a design

-Pinterest, in fact, has a community specially catered to website designs for different devices.

4.

BOOTSTRAP

-A web development-focused website.

-Bootstrap is catered towards providing its users with codes and syntax for web design templates.

-Bootstrap allows you to copy the syntax but you can always just use the codes as inspiration.

-Bootstrap is concerned with creating websites with responsive web design in mind.
